## 3. 转义字符

**3.1 概念**

转义字符以 `\` 开头，用于表示无法直接输入或显示的字符。

**3.2 完整转义字符表**

| 转义序列 | 名称 | 作用 | 示例 |
|----------|------|------|------|
| `\\` | 反斜杠 | 输出一个字面量 `\` | `"C:\\path"` |
| `\'` | 单引号 | 在单引号字符串中嵌入单引号 | `'It\'s'` |
| `\"` | 双引号 | 在双引号字符串中嵌入双引号 | `"He said \"hi\""` |
| `\a` | 响铃 BEL | 触发系统提示音 | `"\a"` |
| `\b` | 退格 BS | 删除前一个字符 | `"abc\bd"` → `abd` |
| `\n` | 换行 LF | 光标移到下一行 | `"a\nb"` |
| `\t` | 水平制表 TAB | 跳至下一个制表位 | `"a\tb"` |
| `\v` | 垂直制表 VT | 垂直方向制表 | 较少使用 |
| `\r` | 回车 CR | 光标回到行首（覆盖） | `"abc\rX"` → `Xbc` |
| `\f` | 换页 FF | 换到新页 | 终端通常无效果 |
| `\ooo` | 八进制转义 | 八进制编码的字符 | `"\141"` → `a` |
| `\xhh` | 十六进制转义 | 十六进制编码的字符 | `"\x61"` → `a` |

**3.3 涉及 Python 特殊写法**
- 八进制 `\ooo` 和十六进制 `\xhh` 转义属于底层编码知识
- `\r` 覆盖行首的特性可以用来做进度条效果

## 4. 原始字符串

**4.1 概念**

在字符串前加 `r` 或 `R`，使字符串内的所有转义字符失效。

**4.2 语法**

```python
r"字符串内容"
r'字符串内容'
```

**4.3 适用场景**

| 场景 | 普通字符串 | 原始字符串 |
|------|-----------|------------|
| Windows 路径 | `"C:\\Users\\name"` | `r"C:\Users\name"` |
| 正则表达式 | `"\\d+\\w+"` | `r"\d+\w+"` |
| LaTeX 公式 | `"\\frac{a}{b}"` | `r"\frac{a}{b}"` |

**4.4 注意事项**
- 原始字符串**末尾不能是奇数个反斜杠**，例如 `r"C:\"` 会报错
- 本质是语法糖，解释器在编译时处理

**4.5 涉及 Python 特殊写法**
- `r""` 前缀是 Python 独有的原始字符串语法
- 并非真的有"原始"数据类型，解释器仍然按正常字符串处理

## 5. 长字符串

**5.1 概念**

用三重引号（三个单引号或三个双引号）包裹的字符串，可以**跨越多行**。

**5.2 语法**

```python
'''这是长字符串'''

"""这也是长字符串"""
```

**5.3 用法**

| 用途 | 示例 |
|------|------|
| 文档字符串（docstring） | 放在模块/类/函数开头 |
| 多行文本块 | 大段文字说明 |
| 多行注释 | 用 `"""` 代替连续多个 `#` |

**5.4 涉及 Python 特殊写法**
- `"""` 既是字符串字面量，又约定俗成用作 docstring
- 如果未赋值给变量，解释器会忽略它（实际生成字符串对象后丢弃）
- 不同于普通注释 `#`，docstring 可以被 `help()` 和 `__doc__` 访问

## 6. 字符串的加法和乘法

**6.1 字符串运算符**

| 运算符 | 操作 | 示例 | 结果 |
|--------|------|------|------|

## 7. 比较运算符

**7.1 基础比较运算符（6个）**

| 运算符 | 含义 | 示例 | 结果 |
|--------|------|------|------|
| `<` | 小于 | `3 < 5` | `True` |
| `<=` | 小于等于 | `3 <= 3` | `True` |
| `>` | 大于 | `5 > 3` | `True` |
| `>=` | 大于等于 | `5 >= 5` | `True` |
| `==` | 等于（比**值**） | `[1,2] == [1,2]` | `True` |
| `!=` | 不等于 | `3 != 5` | `True` |

**7.2 身份运算符（2个）⭐ 重点**

| 运算符 | 含义 | 比较对象 |
|--------|------|----------|
| `is` | 同一对象 | 比较内存地址（`id()` 值） |
| `is not` | 不是同一对象 | 比较内存地址（`id()` 值） |

**7.3 `is` vs `==` 对比**

```python
a = [1, 2, 3]
b = [1, 2, 3]

a == b   # True  — 值相同
a is b   # False — 两个独立对象，地址不同
```

| 对比维度 | `==` | `is` |
|----------|------|------|
| 比较什么 | 值（value） | 对象身份（identity） |
| 底层 | 调用 `__eq__` 方法 | 比较 `id()` |
| 典型用途 | 比较内容是否相同 | 判断是否为同一对象、判断 `None` |

**7.4 Python 小整数缓存 🔑**

Python 启动时会预先缓存 **-5 到 256** 之间的整数对象，所有使用这些值的变量都指向同一个对象：

```python
a = 100; b = 100
a is b   # True   ← 小整数缓存

c = 1000; d = 1000
c is d   # False  ← 超出范围，各自创建新对象
c == d   # True   ← 值仍然相等
```

**7.5 最佳实践**
- 判断 `None`：**始终用 `is`**，不用 `==`
- 比较普通值：用 `==`
- 不要依赖小整数缓存，不同 Python 实现可能范围不同

**7.6 涉及 Python 特殊写法**
- `is`/`is not` 是无法被重载的运算符，比较的是 `id()`
- 小整数缓存是 CPython 的实现细节，PyPy/Jython 行为可能不同

## 8. while循环和break

**8.1 知识点**

| 知识点 | 说明 |
|--------|------|
| `import random` | 导入 random 模块，用于生成随机数 |
| `random.randint(a, b)` | 返回 a 到 b 之间的随机整数（含两端） |
| `while 条件:` | 条件为 `True` 时循环执行，循环次数不固定 |
| `break` | 立即跳出当前循环，不再执行剩余循环体 |
| 计数器递减 | `counts = counts - 1` 控制循环次数 |

**8.2 代码逻辑**

```
生成随机答案 → 设定3次机会 → while 循环（机会>0）→ 猜中就 break 退出 → 没猜中就提示大小并减机会 → 机会用完循环结束
```

**8.3 while 循环结构**

```python
while 条件:
    循环体
    if 某个条件:
        break  # 提前退出
```

| 特点 | 说明 |
|------|------|
| 条件驱动 | 不知道循环次数时使用，由条件控制退出 |
| `break` | 提前终止循环，跳到循环之后的第一条语句 |
| 与 `for` 区别 | `for` 遍历已知序列，`while` 由条件驱动 |

**8.4 涉及 Python 特殊写法**
- `import random` 模块导入，Python 自带丰富的标准库
- `random.randint(1, 10)` 左闭右闭区间（与 `range()` 的左闭右开不同）

## 9. 导入随机数函数及随机数复现

**9.1 知识点**

| 知识点 | 说明 |
|--------|------|
| `random.getstate()` | 获取当前随机数生成器的内部状态 |
| `random.setstate(state)` | 恢复之前保存的随机数状态 |
| 随机数复现 | 保存状态 → 生成随机数 → 恢复状态 → 再次生成相同序列 |
| `for _ in range(n)` | `_` 作占位变量，不关心循环变量的值 |

**9.2 随机数状态机制**

```
第一次生成5个数 → 保存状态 → 中间生成3个数 → 恢复状态 → 再生成5个数（与第一次完全相同）
```

**9.3 `getstate` / `setstate` 用途**

| 场景 | 说明 |
|------|------|
| 调试复现 | 出错时还原相同的随机序列，便于定位问题 |
| 游戏存档 | 保存随机状态，读档后随机结果一致 |
| 实验可重复 | 随机实验保存状态，确保结果可复现 |

**9.4 涉及 Python 特殊写法**

- `_` 作为循环变量名：约定俗成表示"不使用的变量"
- `random.getstate()` / `random.setstate()`：底层状态管理，不常用但很强大
- random 模块的伪随机数由梅森旋转算法生成，状态决定了后续所有随机数

## 10. 数字类型

**10.1 浮点数精度问题**

浮点数在 Python 中用 IEEE 754 双精度存储，无法精确表示某些十进制小数：

```python
>>> 0.1 + 0.2
0.30000000000000004

>>> 0.3 == 0.1 + 0.2
False
```

累加时误差会不断放大，不适合精确计数或等值比较。

**10.2 使用 `decimal.Decimal` 解决精度问题**

```python
import decimal

a = decimal.Decimal('0.1')   # 用字符串传入，避免 float 精度损失
b = decimal.Decimal('0.2')
print(a + b)                 # 0.3
print(a + b == decimal.Decimal('0.3'))  # True
```

> ⚠️ 不能直接用 float 创建 Decimal：`decimal.Decimal(0.1)` 仍会继承浮点误差。

**10.3 复数（Complex Numbers）**

Python 原生支持复数，使用 `j` 表示虚部：

```python
x = 1 + 2j
x.real   # 1.0  实部
x.imag   # 2.0  虚部

(1 + 2j) + (3 + 4j)  # (4+6j)
(1 + 2j) * (3 + 4j)  # (-5+10j)
```

**10.4 涉及 Python 特殊写法**

| 特殊点 | 说明 |
|--------|------|
| 浮点精度 | IEEE 754 双精度的固有限制，不是 Python bug |
| `Decimal` | 用字符串构造才能保证精度，`Decimal('0.1')` |
| 复数原生支持 | `1+2j` 是合法的字面量，`.real` / `.imag` 直接访问 |
| 复数运算 | 复数天然支持 `+` `-` `*` `/`，不需要额外库 |

## 11. import 的两种写法

**11.1 两种导入方式**

| 写法 | 示例 | 调用方式 |
|------|------|----------|
| `from 模块 import 类` | `from decimal import Decimal` | 直接 `Decimal('0.1')` |
| `import 模块` | `import decimal` | 带前缀 `decimal.Decimal('0.1')` |

**11.2 与 Java 的类比**

| Python | Java 等效 | 特点 |
|--------|-----------|------|
| `from decimal import Decimal` | `import java.math.BigDecimal;` | 简洁，直接用类名 |
| `import decimal`<br>`decimal.Decimal(...)` | `new java.math.BigDecimal(...)`（全限定名） | 明确来源，安全 |

**11.3 Python 通配符导入（不推荐）**

```python
from decimal import *  # ❌ PEP 8 反对，污染命名空间
```

对应 Java 的 `import java.math.*;`，同样不推荐。

**11.4 最佳实践**

| 语言 | 推荐写法 | 原因 |
|------|----------|------|
| Python | `from decimal import Decimal` | 简洁、清晰、官方文档常用 |
| Java | `import java.math.BigDecimal;` | 明确、高效、IDE 自动优化 |

**11.5 记忆口诀**
- `from X import Y` → "从 X 包里拿 Y 类" → Java 的 `import X.Y;`
- `import X` → "加载整个 X 模块" → Java 的全限定名调用

## 12. 数字运算

**12.1 算术运算符**

| 运算符 | 含义 | 示例（x=10, y=3） | 结果 |
|--------|------|-------------------|------|
| `+` | 加法 | `x + y` | `13` |
| `-` | 减法 | `x - y` | `7` |
| `*` | 乘法 | `x * y` | `30` |
| `/` | 除法（浮点） | `x / y` | `3.333...` |
| `//` | 地板除（向下取整） | `x // y` / `-3 // 2` | `3` / `-2` |
| `%` | 取余 | `x % y` | `1` |
| `**` | 幂运算 | `x ** y` | `1000` |
| `-x` | 相反数 | `-x` | `-10` |
| `+x` | 正号（本身） | `+x` | `10` |
| `abs(x)` | 绝对值 | `abs(-10)` | `10` |

**12.2 地板除 `//` 详解**

```python
10 // 3   # 3   正常向下取整
-3 // 2  # -2  向下取整，-1.5 → -2（比目标小的最大整数）
```

**12.3 `divmod()` — 同时返回商和余数**

```python
divmod(10, 3)  # (3, 1)

# 用途1：秒数转 时:分:秒
minutes, seconds = divmod(3725, 60)
hours, minutes = divmod(minutes, 60)
# → 1:02:05

# 用途2：进制转换
n, r = divmod(n, base)  # r 是当前位的值

# 用途3：分页计算
full_pages, last_page_items = divmod(total, page_size)
```

**12.4 `pow()` 幂运算**

```python
pow(x, y)      # x ** y
pow(2, 3, 5)   # 2**3 % 5 = 3，支持第三个参数取模
```

**12.5 复数操作**

```python
c = complex(3, 4)     # (3+4j)
c.conjugate()         # (3-4j) 共轭复数
```

**12.6 涉及 Python 特殊写法**
- Python 除法 `/` 始终返回浮点数（不同于 Java/C 的整数除法）
- `//` 是真正的"地板除"，负数向下取整
- `divmod()` 一行同时拿到商和余数
- `pow(x, y, z)` 三参数形式比 `x**y % z` 更高效

## 13. bool 布尔类型

`bool(x)` 将任意对象转换为布尔值（`True` 或 `False`）。

**13.5 涉及 Python 特殊写法**

## 15. 短路逻辑和运算符优先级

**15.1 短路逻辑（Short-circuit Evaluation）**

`and`：左边为 `False` → 短路，直接返回左边的值，不计算右边
`or`：左边为 `True` → 短路，直接返回左边的值，不计算右边

```python
0 and 1       # 0   ← 左边假，短路返回0
3 and 4       # 4   ← 左边真，继续返回右边
3 or 4        # 3   ← 左边真，短路返回3
0 or 4        # 4   ← 左边假，继续返回右边
7 and 8 and 9 # 9   ← 连续 and，返回最后一个
```

**15.2 复杂表达式分析**

```python
(not 1) or (0 and 1) or (3 and 4) or (5 and 6) or (7 and 8 and 9)
#  0    or   0       or    4      or    6      or       9
# → 4  （or 遇到第一个真值就返回）
```

**15.3 运算符优先级表（由低到高）**

| 优先级 | 运算符 | 描述 |
|--------|--------|------|
| 1 | `lambda` | Lambda 表达式 |
| 2 | `if-else` | 条件表达式 |
| 3 | `or` | 布尔或 |
| 4 | `and` | 布尔与 |
| 5 | `not x` | 布尔非 |
| 6 | `in, not in, is, is not, <, <=, >, >=, !=, ==` | 成员/身份/比较 |
| 7 | `\|` | 按位或 |
| 8 | `^` | 按位异或 |
| 9 | `&` | 按位与 |
| 10 | `<<, >>` | 移位 |
| 11 | `+, -` | 加法、减法 |
| 12 | `*, @, /, //, %` | 乘法、除法、地板除、取余 |
| 13 | `+x, -x, ~x` | 正号、负号、按位翻转 |
| 14 | `**` | 指数 |
| 15-17 | 下标/切片/调用/属性/显示 | 基础操作 |

> 💡 实际工作中加括号最可靠，不需要死记优先级。

**15.4 涉及 Python 特殊写法**
- 短路逻辑是 Python 的核心机制，`and`/`or` 返回值而非布尔值
- `or` 可用于提供默认值：`name = input_name or "匿名"`
- `and` 可用于链式判断：`user and user.name`

## 16. 分支 if

**16.1 多重 if-elif-else**

```python
if condition1:
    statement(s)
elif condition2:
    statement(s)
elif condition3:
    statement(s)
else:
    # 所有条件均不成立时
```

**16.2 一行 if-else 表达式**

```python
# 普通写法
if age < 18:
    print("禁止访问")
else:
    print("欢迎")

# 一行写法
print("禁止访问") if age < 18 else print("欢迎")
```

**16.3 多条件一行表达式（链式）**

```python
score = 66
level = ('D' if 0 <= score < 60 else
         'C' if 60 <= score < 80 else
         'B' if 80 <= score < 90 else
         'A' if 90 <= score < 100 else
         'S' if score == 100 else
         "请输入 0~100 之间的分值")
```

**16.4 if 嵌套**

```python
if age < 18:
    print("禁止访问")
else:
    if isMale:
        print("任君选购！")
    else:
        print("抱歉，可能不适合~")
```

**16.5 涉及 Python 特殊写法**
- 一行 `if-else` 表达式（三元运算符）：`X if 条件 else Y`
- 链式比较：`0 <= score < 60` 是 Python 专有语法
- 链式 if-else 可实现类似 `switch-case` 的效果

## 17. while循环

**17.1 `while` + `break`**

```python
while True:
    answer = input("可以退出循环了吗？")
    if answer == "可以！":
        break          # 跳出循环
    print("哎，好累~~~")
```

**17.2 `continue` — 跳过本次，进入下一次循环**

```python
i = 0
while i < 10:
    i += 1
    if i % 2 == 0:
        continue       # 跳过偶数
    print(i)           # 只输出奇数：1 3 5 7 9
```

**17.3 `while-else` — 循环正常结束后执行**

```python
day = 1
while day <= 7:
    answer = input("今天有好好学习吗？")
    if answer != "有":
        break          # break 退出 → else 不会执行
    day += 1
else:
    print("非常棒，你已经坚持了7天连续学习！")  # 只有没 break 才执行
```

> ⚠️ 如果循环被 `break` 中途退出，`else` 不会执行。

**17.4 循环嵌套**

```python
# 九九乘法表
for i in range(1, 10):
    for j in range(1, i + 1):
        print(f"{j}×{i}={i*j}", end="\t")
    print()
```

**17.5 涉及 Python 特殊写法**
- `while-else` 是 Python 独有语法，`else` 在循环**正常结束**（非 `break`）时执行
- `continue` 跳过本次循环剩余代码，进入下一次循环
- `break` 跳出**当前这一层**循环，外层循环不受影响

## 18. for 循环

**18.1 遍历字符串**

```python
for each in "FishC":
    print(each)      # 逐字符输出 F i s h C
```

**18.2 `range()` — 与 for 配合使用**

| 用法 | 说明 | 示例 | 结果 |
|------|------|------|------|
| `range(stop)` | 从 0 到 stop-1 | `list(range(5))` | `[0,1,2,3,4]` |
| `range(start, stop)` | 从 start 到 stop-1 | `list(range(2, 7))` | `[2,3,4,5,6]` |
| `range(start, stop, step)` | 步长为 step | `list(range(1,10,2))` | `[1,3,5,7,9]` |
| 反向 | step 为负 | `list(range(10,0,-1))` | `[10,9,...,1]` |

> ⚠️ `for i in 1000000` 会报错，整数不可迭代，必须用 `range()`。

**18.3 `for-else` — 循环正常结束后执行**

```python
for n in range(2, 10):
    for x in range(2, n):
        if n % x == 0:
            print(n, "=", x, "*", n // x)
            break
    else:
        print(n, "是一个素数")    # 内层循环没 break 才执行
```

**18.4 涉及 Python 特殊写法**
- `for-else` 是 Python 独有语法，`else` 在循环**正常结束**（非 `break`）时执行
- `range()` 是左闭右开区间，返回可迭代对象而非列表
- Python 中 `for` 遍历的是可迭代对象，不是 C/Java 风格的三段式 for

## 19. 列表和列表的切片

**19.1 列表的特点**

```python
rhyme = [1, 2, 3, 4, 5, "上山打老虎"]  # 可存不同类型数据
```

**19.2 索引访问**

| 索引 | 含义 | 示例 | 结果 |
|------|------|------|------|
| `[0]` | 第一个元素 | `rhyme[0]` | `1` |
| `[1]` | 第二个元素 | `rhyme[1]` | `2` |
| `[-1]` | 最后一个元素 | `rhyme[-1]` | `'上山打老虎'` |
| `[-2]` | 倒数第二个 | `rhyme[-2]` | `5` |
| `len()` | 获取列表长度 | `len(rhyme)` | `6` |

**19.3 切片 `[start:stop:step]`**

| 写法 | 含义 | 示例 | 结果 |
|------|------|------|------|
| `[0:3]` | 索引0~2（不含3） | `rhyme[0:3]` | `[1, 2, 3]` |
| `[3:6]` | 索引3~5 | `rhyme[3:6]` | `[4, 5, '上山打老虎']` |
| `[:3]` | 开头到索引2 | `rhyme[:3]` | `[1, 2, 3]` |
| `[3:]` | 索引3到结尾 | `rhyme[3:]` | `[4, 5, '上山打老虎']` |
| `[:]` | 整个列表**副本** | `rhyme[:]` | 完整复制 |
| `[::2]` | 步长为2 | `rhyme[::2]` | `[1, 3, 5]` |
| `[::-1]` | 整个列表**倒序** | `rhyme[::-1]` | `['上山打老虎',5,4,3,2,1]` |
| `[::-2]` | 倒序+步长2 | `rhyme[::-2]` | 从末尾隔一个取 |

**19.4 涉及 Python 特殊写法**
- 负索引：`-1` 直接取最后一个元素，不需要 `len()-1`
- 切片 `[:]` 是浅拷贝，生成新列表
- 切片区间是**左闭右开** `[start, stop)`
- `[::-1]` 一行翻转整个序列

## 20. 列表的添加和删除

**20.1 添加元素**

| 方法 | 说明 | 示例 | 结果 |
|------|------|------|------|
| `append(x)` | 末尾追加一个元素 | `heros.append("黑寡妇")` | 添加单个 |
| `extend(iter)` | 末尾追加多个元素 | `heros.extend(["鹰眼","灭霸"])` | 批量添加 |
| `insert(i, x)` | 在位置 i 插入 | `s.insert(1, 2)` | `[1,2,3,4,5]` |
| `s[len(s):] = [x]` | 切片方式追加 | `s[len(s):] = [6]` | 等价 append |

**20.2 删除元素**

| 方法 | 说明 | 示例 |
|------|------|------|
| `remove(x)` | 删除第一个匹配的**值** | `heros.remove("灭霸")` |
| `pop(i)` | 删除索引 i 的元素，**返回该元素** | `heros.pop(2)` |
| `clear()` | 清空所有元素 | `heros.clear()` |

> ⚠️ `remove(x)` 删除不存在的元素会抛出 `ValueError`。

**20.3 添加/删除对比总结**

| 操作 | 方法 | 返回值 | 说明 |
|------|------|--------|------|
| 尾加一个 | `append(x)` | `None` | 修改原列表 |
| 尾加多个 | `extend([...])` | `None` | 接收可迭代对象 |
| 指定位置插入 | `insert(i, x)` | `None` | 在 i 之前插入 |
| 按值删除 | `remove(x)` | `None` | 只删第一个匹配 |
| 按索引删除 | `pop(i)` | 被删元素 | 不传参默认删最后一个 |
| 清空 | `clear()` | `None` | 等价 `del s[:]` |

**20.4 涉及 Python 特殊写法**
- `s[len(s):] = [x]`：切片操作可以直接赋值修改列表
- `pop()` 默认删除**最后一个**元素（栈操作）
- `clear()` 清空列表本身，但变量引用不变
